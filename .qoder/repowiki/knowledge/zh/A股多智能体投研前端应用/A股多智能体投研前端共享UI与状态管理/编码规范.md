- 自定义 Hook 命名以 `use` 开头并严格遵循关注点分离，如 `useSSE` 处理流式连接，`useAnalysisJobRecovery` 处理状态恢复。
- API 请求统一通过 `services/api.ts` 中的 `ApiService` 类实例发起，自动注入 `Authorization` Bearer Token 并处理 JSON 响应解析。
- 组件样式采用 Tailwind CSS，普遍使用 `dark:` 前缀实现深色模式适配，并利用 `backdrop-blur` 和透明度类构建毛玻璃视觉效果。
- Store 动作（Actions）设计为细粒度更新函数（如 `updateAgentStatus`, `addReportChunk`），避免在组件内直接操作状态，保持状态变更的可预测性。

- **统一网关与路由**：`main.py` 集中定义所有 HTTP 端点，根据业务域委托给 `services/` 下的各子模块处理，实现逻辑解耦。
- **共享数据基础设施**：`database.py` 提供全局 SQLAlchemy 引擎与会话管理，所有子模块通过统一的 `get_db_ctx` 上下文管理器访问持久化层，确保事务一致性。
- **异步任务编排**：内置基于内存/Redis 的 `job_store` 管理长耗时分析任务，通过 `ThreadPoolExecutor` 和 `asyncio` 协调后台作业，避免阻塞主事件循环。
- **全局状态与缓存**：维护跨请求共享的 `DataCollector` 实例与 A 股名称映射缓存，优化多模块间的数据获取效率并减少外部 API 调用。
